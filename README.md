# AI-Powered Job Application Tracker

## Overview
A full-stack job tracking system built with Java + Spring Boot...

## Features
- JWT Authentication
- Groq AI skill gap analysis  
- REST API with MySQL backend
- HTML/Bootstrap/JS dashboard

## Tech Stack
Java, Spring Boot, MySQL, JWT, Groq AI API, HTML, Bootstrap, JavaScript

## Setup Instructions
1. Clone the repo
2. Configure application.properties (DB credentials, Groq API key)
3. Run: mvn spring-boot:run
4. Open: http://localhost:8080
